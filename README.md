# Analisis_de_Viajes_en_Chicago_-Zuber-_Proyecto
Sprint 8 Proyecto

El objetivo de este proyecto fue analizar datos reales de viajes en taxi en Chicago para identificar patrones de actividad entre compañías, determinar los barrios con mayor volumen de finalizaciones de viaje y evaluar si las condiciones climáticas influyen en la duración de los trayectos desde el Loop hasta el Aeropuerto Internacional O’Hare. El análisis combina técnicas de Exploratory Data Analysis (EDA) y estadística inferencial para responder preguntas clave sobre movilidad urbana.

## Tecnologias Utilizadas
- Python, pandas, NumPy
- matplotlib, seaborn
- SQL
- Limpieza y preparación de datos
- EDA (Exploratory Data Analysis)
- Visualización de datos
- Pruebas de hipótesis (estadística inferencial)
- Jupyter Notebook
